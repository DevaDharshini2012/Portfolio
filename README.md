the # Deva Dharshini K - Portfolio

This is a modern, responsive portfolio website built using React, HTML, and CSS. It uses a "No-Build" approach, meaning you don't need Node.js or NPM installed to run it.

## How to Run

### Quick Start
1. Ensure you have **Python** installed on your system.
2. Open a terminal (PowerShell or Command Prompt).
3. Navigate to this folder:
   ```powershell
   cd "c:\Users\kdeva\OneDrive\Desktop\portfolio"
   ```
4. Start a local development server:
   ```powershell
   python -m http.server 8000
   ```
5. Open your web browser and go to:
   [http://localhost:8000](http://localhost:8000)

### Alternative Method
You can also simply double-click the `index.html` file to open it in your browser. However, using a local server (like the Python method above) ensures that all features (like fonts and icons) load reliably.

## Project Structure
- `index.html`: Contains the HTML structure and the React component logic.
- `styles.css`: Contains the premium design system and styling tokens.
- `README.md`: These instructions.
